What is a promise?

A real-life example

  • you make an order in a cafe
  • the order is being prepared, you are waiting for it
  • The order was completed successfully
  • there was a breakdown, the order is not ready

Promise in javaScript

A Promise is simply an object in JavaScript

    A promise is always in one of the three states

  • pending : which is initial state, neither fulfilled nor rejected
  • fulfilled : meaning that the operation completed successfully
  • rejected: meaning that the operation failed

Once a promise is fulfilled or rejected,it cannot switch to any other state.

    Why promise?
  • Promises make asynchronous code easier to handle than callbacks.
  • With promises we can avoid the hell of callbacks.

How to create a Promise?

To create a promise, we need to create an instance of the Promises class.

Then pass a callback function to the constructor.

The callback function accepts two functions as a parameter

Then

The first argument to the .then method is a function that is executed when the promise enters the "successful" state and receives the result.

The second argument to .then is a function that is executed when the promise enters the "failed" state and receives an error.

Catch

The first argument to the .then method is a function that is executed when the promise enters the "successful" state and receives the result.

finally

the code will be executed in any case when the promise is completed: successfully or with an error.

The handler called from finally has no arguments. In finally, we don't know how the promise was completed. And this is fine, because usually our task is to perform "general" final procedures.
